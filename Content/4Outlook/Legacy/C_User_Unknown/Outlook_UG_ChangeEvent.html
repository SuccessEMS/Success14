Changing an Event

You can edit meetings to add services, change or add rooms, and so on. See Also: Booking a One-Time Meeting and Booking a Recurrent Meeting

  1. From Microsoft® Outlook, open the reservation you wish to change.

Tip: You can edit the Start Time and/or End Time in the Microsoft® Outlook calendar.

  1. If the event is recurring, an Open Recurring Item message opens. 

To edit the Start Time and/or End Time, or any details for just one booking in the series, leave Open this occurrence selected, and then click OK.
To edit details other than the Start Time and/or End Time for all bookings in the series, select The entire series, and then click OK.

  1. Click the EMS for Outlook icon in the Microsoft® Outlook toolbar.
  1. The EMS for Outlook add-in opens on the Selected Rooms tab so you can see rooms associated with the booking, change meeting details, work with services, or work with space, or you can click Delete to cancel the event. If you are editing a single event in a recurring meeting, you can modify the start and end times.
  2. Follow the steps below to make changes.
To change headcount
Click the Selected Rooms tab. Click on the number in the Attendance column that you wish to change and make your edits.

To add a room
Click the List tab and click the Add icon next to available rooms to add them.
Tip: Click the Information icon to see more information about a room.
To remove a room
From the Selected Rooms tab, click the Remove icon. Bookings that have been canceled or occur in the past will be grayed out and unavailable for removal. When you remove a room from a recurrent reservation, you may be prompted that all selected bookings in that reservation with that room will be canceled; this indicates only the rooms for which you have clicked the Remove icon.

To change services
Click the Services tab. Under the Bookings area in the left panel, you will see each event listed by date (recurring meetings show rooms as bookings by multiple dates; meetings with more than one room show each room as a booking, but all have the same date); you can work with existing services by expanding the booking date and selecting the service. To add more services, select the booking and choose from categories of services in the center panel. Then finalize details in the right panel.
 
  1. Click Save/Send to save your changes.

Note: Scheduling changes only show in the Microsoft® Outlook and EMS Desktop Client calendars once the user has finished making changes and closed the EMS for Outlook add-in window.
